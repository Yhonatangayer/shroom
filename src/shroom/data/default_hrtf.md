# Default HRTF

The default HRTF file (`default_hrtf.sofa`) included in this package is the **Neumann KU 100** dummy head HRTF compilation.

## Reference

[48] B. Bernschütz, “A spherical far field HRIR/HRTF compilation of the neumann KU 100,” in Proceedings of the 40th Italian (AIA) annual conference on acoustics and the 39th German annual conference on acoustics (DAGA) conference on acoustics. Merano, Italy: AIA/DAGA, 2013, p. 29.

## Source
This dataset is widely used in acoustic research and is available in the SOFA format.
